# Rust Runtime Example

An example implemention of a DICE runtime utilizing the Rust programming language, demonstrating how to build and test an input artifact in Rust.

## Dependencies

1. zsh

1. wasm-opt (`brew install binaryen`)

1. wasmer

1. Rust

1. Make

## Commands

### Clean

`make clean`

### Build

`make build`

### Test

`make test`

### Inspect

`make inspect`
